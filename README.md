# Nashville Housing Data Cleaning Project (MySQL)

## Project Overview
This project focuses on cleaning, transforming, and preparing raw housing data using MySQL.
The objective was to improve data quality, standardize formats, remove duplicates,
and optimize the dataset for analysis and reporting purposes.


The project demonstrates my practical SQL skills and learning commonly used in data analytics and business intelligence workflows.

## Dataset
The dataset used for this project contains housing records with issues such as:
- Inconsistent date formats
- Duplicate records
- Renaming of data column 
- Unstandardized column names
- Extra spaces in text fields

## SQL Skills & Concepts Demonstrated

### Database Management
- ALTER TABLE
- UPDATE statements
- Column renaming
- Data type modification

### Data Cleaning & Transformation
- TRIM()
- STR_TO_DATE()
- ALTER
- Standardizing data formats

### Advanced SQL Techniques
- Common Table Expressions (CTEs)
- Window Functions using ROW_NUMBER()
- Duplicate detection and removal

## Key Tasks Performed
- Renamed improperly formatted column names
- Converted date fields into standard SQL date format
- Removed leading and trailing spaces from text fields
- Identified and removed duplicate records
- Improved dataset consistency and readability
- Prepared clean data for analysis and visualization
